# Axiom Clash - Tactical RPG Battle

## Description
Axiom Clash is a pixel-art turn-based tactical RPG where you assemble a party of three legendary heroes to defend the kingdom from an onslaught of monsters. Featuring structured turn handling, stat calculations, and an initiative gamble system.

## Story
The Kingdom of Axiom has lived in peace for centuries, protected by the ancient seals. But now, the seals have weakened, and monsters from the void are pouring into the realm. Seeing her people in peril, the Queen has summoned the six legendary heroes of Axiom. These heroes, each with unique abilities and backgrounds, must unite to push back the darkness and seal the void once and for all. As their commander, you must lead them through strategic battles, choosing the right party for every encounter.

## Setup Instructions
1. Download the `AxiomClash.html` file and the `assets` folder.
2. Ensure the `assets` folder is in the same directory as `AxiomClash.html`.
3. Open `AxiomClash.html` in any modern web browser (Chrome, Firefox, Edge).
4. Click the "START GAME" button on the initial screen to enable audio.

## Controls
- **Mouse:** Click buttons and heroes to interact.
- **Keyboard Navigation:**
    - **WASD / Arrow Keys:** Navigate menus and targets.
    - **Enter / Space:** Confirm selection.
    - **Esc:** Toggle Pause menu.
- **Battle Controls:**
    - **Strike:** Basic attack.
    - **Skills:** Use character-specific abilities (costs MP).
    - **Info:** View detailed stats of all combatants.
